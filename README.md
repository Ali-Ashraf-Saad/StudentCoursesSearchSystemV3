# نظام الاستعلام عن المقررات الدراسية مع لوحة تحليلات

نظام ويب متكامل يتيح للطلاب الاستعلام عن مقرراتهم الدراسية، وجداول الامتحانات، والمواد التعليمية، بالإضافة إلى لوحة تحليلات لحظية للمسؤولين. النظام يعمل بـ **PHP** مع تخزين قائم على ملفات JSON، وتتبع حي للمستخدمين، ورسوم بيانية تفاعلية بواسطة **Plotly**.

---

## ✨ المميزات

### للطلاب (الواجهة الرئيسية – `index.php`)

- **البحث** بالرقم الأكاديمي أو الاسم (مع تطبيع الحروف العربية والبحث الجزئي)
- عرض المقررات المسجلة مع:
  - اسم المقرر والكود
  - لجنة الامتحان، القاعة، اليوم، التاريخ، الفترة والزمن
  - روابط مباشرة **Google Drive** لمحتوى المقرر (حيثما توجد)
- **سجل البحث** (مخزن في localStorage بالمتصفح) – يمكن النقر لإعادة البحث
- **تصدير** قائمة مقررات الطالب كصورة PNG (باستخدام `html2canvas`)
- عداد زوار فوري (عبر `counter.php`)

### للمسؤولين / الموظفين

- **لوحة إحصائيات مخفية** – اضغط على اسم المطور "Ali Ashraf" 10 مرات لفتحها
- مقاييس فورية:
  - المستخدمون النشطون الآن (نبضة كل 3 ثوانٍ)
  - عدد المستخدمين الذين يفتحون لوحة الإحصائيات حاليًا
  - إجمالي مرات فتح لوحة التحكم ولوحة الإحصائيات
  - إجمالي عمليات البحث، بحوث اليوم، متوسط البحث لكل ساعة / لكل مستخدم
  - الطلاب الفريدين الذين تم البحث عنهم، ساعة الذروة، إجمالي مرات تصدير الصور
- **جدول أعلى 5 طلاب تم البحث عنهم**
- **سجل بحث حي** (آخر 50 استعلام مع التوقيت والـ IP)
- **جدول جميع الطلاب وعدد مرات البحث** مع زر لإظهار أكثر من 5 مدخلات
- **زر تصفير جميع الإحصائيات** (يمسح العدادات والسجلات والمستخدمين النشطين)

### لوحة التحليلات (`dashboard.html`)

- **أربع بطاقات مقاييس رئيسية**: إجمالي الطلاب، المقررات الفريدة، متوسط المقررات لكل طالب، المقرر الأعلى
- **رسوم بيانية تفاعلية** (Plotly):
  - أعلى 15 مقرر من حيث التسجيل (شريطي)
  - توزيع المقررات (دائري)
  - توزيع عدد المقررات لكل طالب (هستوجرام)
  - مخطط صندوقي لتوزيع التسجيلات
- **ترتيب شعبية المقررات** بأشرطة متحركة
- تصميم متجاوب، واجهة داكنة (Dark Theme) وحديثة

### تتبع المستخدمين الحي

- `active.php` – نقطة نهاية النبضة (`client_id` مخزن في localStorage)
- يتتبع حالتين:
  - المستخدمين النشطين عاديًا (مهلة 20 ثانية)
  - المستخدمين الذين فتحوا لوحة الإحصائيات (علامة `stats=1`)
- يستخدم في كل من الصفحة الرئيسية ولوحة التحكم لعرض المستخدمين المتصلين الآن

## 🚀 دليل الاستخدام

### المستخدم العادي (طالب / زائر)

1. افتح `index.php`.
2. اكتب **الرقم الأكاديمي** أو **الاسم** (عربي) في مربع البحث.
3. تظهر النتائج تلقائيًا مع المقررات وتفاصيل الامتحانات وروابط Drive.
4. انقر على زر الكاميرا لتحميل صورة قائمة مقررات الطالب.
5. يتم حفظ عمليات البحث الأخيرة في لوحة السجل – انقر على أي منها لتكرار البحث.

### فتح لوحة الإحصائيات المخفية

1. في `index.php`، انتقل إلى أسفل الصفحة (منطقة التذييل).
2. **اضغط على "Ali Ashraf" (اسم المطور) 10 مرات متتالية**.
3. ستظهر لوحة الإحصائيات مع بيانات حية.
4. يتم عد كل مستخدم يفتح هذه اللوحة بشكل منفصل (`stats_active_users`).
5. استخدم زر **"🗑️ تصفير جميع الإحصائيات"** لإعادة تعيين جميع العدادات والسجلات.

### لوحة التحليلات

- افتح `dashboard.html` (أو اضغط على زر Dashboard العائم في `index.php`).
- تعرض لوحة التحكم مقاييس تحليلية عالية المستوى بناءً على `students.json` و `courses.json`.
- الرسوم البيانية تفاعلية – يمكن التحويم والتكبير/التصغير وتحريك المؤشر.
- البيانات تُحدث عند إعادة تحميل الصفحة (لا تحديث تلقائي في هذا الإصدار).

## 🔄 نقاط النهاية API (للمطورين)

| النقطة                       | الطريقة | الوصف                                                                     |
|------------------------------|---------|---------------------------------------------------------------------------|
| `search.php?q=<استعلام>&commit=1&client_id=<id>` | GET | إرجاع الطلاب المطابقين. `commit=1` يسجل عملية البحث (زيادة العداد، التسجيل في السجل، search_counts). |
| `active.php?client_id=<id>[&stats=1/0]`       | GET | تسجيل نبضة. `stats=1` تعني أن المستخدم يشاهد لوحة الإحصائيات.          |
| `counter.php?action=increment`               | GET   | زيادة وإرجاع إجمالي عدد الزوار.                                            |
| `dashboard_data.php`                         | GET   | إرجاع الإحصائيات المجمعة للوحة التحكم.                                   |
| `reset_stats.php`                            | POST  | تصفير جميع العدادات والسجلات (يتطلب POST).                                |
| `export_counter.php`                         | GET   | زيادة عداد تصدير الصور.                                                   |
| `dashboard_page_open.php` / `stats_open.php` | GET   | زيادة عداد فتح كل من لوحة التحكم ولوحة الإحصائيات.                        |

جميع الاستجابات بصيغة `application/json`.
